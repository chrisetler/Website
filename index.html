Projects

Here are some of my more presentable projects. For a more comprehensive list, feel free to check out my github page.
Todo List
A Java based todo list application that allows the user to specify a description, date and priority for items. Items are autosaved to a file that is loaded on startup, and can be imported and exported. Future updates will allow for importing and exporting in a more standardized manner, using something more widely supported, such as .CSV.
(github)
Mandelbrot Viewer
A mandelbrot viewer written in C++ using freeglut and openGL headers. The fractal is only re-rendered refreshed by pressing F5 or Space, making it easy to pan and zoom without having to wait for it to reload. Controls: Refresh - F5 or Space, Color Modes - 1-6, Pan - up,down,left,right , Zoom - (+/-)
(github)
Image Steganography Applet
A web applet written in ASP.NET and C# that allows one to hide an image inside another image using a concept known as steganography. I wrote this project as a way to familiarize myself with the basics of ASP.NET and html.
(live) (github)
Huffman Coding Text Compression Applet
A web applet written in ASP.NET and C# that takes in a snippet of text and returns a corresponding dictionary generated using a Huffman Coding algorithm, which when used to encode the text would reduce the file size almost down to the entropy of the text. Huffman Coding is a very general purpose compression method that is typically used after all other forms of compression (which reduce the entropy) to store the file in its low entropy format.
(live) (github)
Huffman Coding Text (in c)
This is a C program that performs the same function as the ASP.NET Huffman Coding applet, but runs much faster.
(github)
